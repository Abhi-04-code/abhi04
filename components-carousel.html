BOOTSTRAP CAROUSEL

▶ Carousel Overview

Bootstrap carousels are slideshows for cycling through elements—images, text, or custom markup. They include built-in support for previous/next controls and indicators, with optional auto-rotation and touch/swipe support.

Pro Tip: Carousels are perfect for showcasing featured content, product images, testimonials, or any content that benefits from a slideshow presentation.

<!-- Basic Carousel Structure -->
<div id="basicCarousel" class="carousel slide" data-bs-ride="carousel">
  <!-- Indicators -->
  <div class="carousel-indicators">
    <button type="button" data-bs-target="#basicCarousel" data-bs-slide-to="0" class="active"></button>
    <button type="button" data-bs-target="#basicCarousel" data-bs-slide-to="1"></button>
    <button type="button" data-bs-target="#basicCarousel" data-bs-slide-to="2"></button>
  </div>
  
  <!-- Slides -->
  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="slide1.jpg" class="d-block w-100" alt="First slide">
      <div class="carousel-caption">
        <h5>First Slide</h5>
        <p>Some description</p>
      </div>
    </div>
    <div class="carousel-item">
      <img src="slide2.jpg" class="d-block w-100" alt="Second slide">
    </div>
  </div>
  
  <!-- Controls -->
  <button class="carousel-control-prev" data-bs-target="#basicCarousel" data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </button>
  <button class="carousel-control-next" data-bs-target="#basicCarousel" data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>

  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="slide1.jpg" class="d-block w-100" alt="...">
    </div>
    <div class="carousel-item">
      <img src="slide2.jpg" class="d-block w-100" alt="...">
    </div>
  </div>
</div>

<!-- JavaScript Control -->
<script>
const carousel = new bootstrap.Carousel('#autoCarousel')
document.getElementById('toggleAutoPlay').addEventListener('click', function() {
  if (this.textContent === 'Pause Auto-play') {
    carousel.pause()
    this.textContent = 'Start Auto-play'
  } else {
    carousel.cycle()
    this.textContent = 'Pause Auto-play'
  }
})
</script>

<div class="carousel-thumbnails">
  <img src="thumb1.jpg" class="active" data-bs-target="#thumbCarousel" data-bs-slide-to="0">
  <img src="thumb2.jpg" data-bs-target="#thumbCarousel" data-bs-slide-to="1">
  <img src="thumb3.jpg" data-bs-target="#thumbCarousel" data-bs-slide-to="2">
</div>

<script>
// Update active thumbnail
const thumbCarousel = document.getElementById('thumbCarousel')
thumbCarousel.addEventListener('slide.bs.carousel', function (e) {
  const thumbnails = document.querySelectorAll('.carousel-thumbnails img')
  thumbnails.forEach((thumb, index) => {
    thumb.classList.toggle('active', index === e.to)
  })
})
</script>

▶ Carousel Options

Carousel Classes and Data Attributes

Class/Attribute Description Usage
.carousel Base carousel container Required for all carousels
.carousel-inner Container for slides Holds all carousel items
.carousel-item Individual slide Each piece of content in carousel
.carousel-fade Fade transition Adds crossfade animation
data-bs-ride Auto-start carousel "carousel" to auto-start
data-bs-interval Slide interval Milliseconds between slides
data-bs-pause Pause on hover "hover" to pause on mouseover

Carousel Best Practices

  • Optimize Images: Use properly sized and compressed images for fast loading.
  • Clear Navigation: Provide visible controls and indicators for user interaction.
  • Accessible Content: Include proper alt text and ensure keyboard navigation works.
  • Reasonable Timing: Use appropriate auto-advance intervals (3-5 seconds recommended).
