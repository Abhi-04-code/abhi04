▶ Introduction to Bootstrap Utilities

Bootstrap utilities are single-purpose CSS classes that allow you to style elements directly in your HTML without writing custom CSS. They follow a consistent naming convention and work responsively across all breakpoints.

Why Use Utilities? Utilities enable rapid prototyping, reduce CSS bloat, ensure consistency, and make your code more maintainable.

Utility Class Structure

Most utilities follow this pattern: {property}{sides}-{size} or {property}-{value}

Examples:
.mt-3    /* margin-top: 1rem */
.pb-4    /* padding-bottom: 1.5rem */
.text-center
.bg-primary
.d-flex
.border
.shadow
.w-50    /* width: 50% */
.fs-4    /* font-size: 1.5rem */
Breakpoint Syntax:
/* Small devices (≥576px) */
.sm-{utility}

/* Medium devices (≥768px) */
.md-{utility}

/* Large devices (≥992px) */
.lg-{utility}

/* Extra large (≥1200px) */
.xl-{utility}

/* Extra extra large (≥1400px) */
.xxl-{utility}

/* Example: */
.d-none .d-md-block

▶ Quick Reference Cheat Sheet

Spacing

  • .m{t|r|b|l|x|y}-{0|1|2|3|4|5|auto}
  • .p{t|r|b|l|x|y}-{0|1|2|3|4|5}

Display

  • .d-{none|inline|block|flex|grid}
  • .d-{print}-{value}

Flex

  • .flex-{row|column|wrap|nowrap}
  • .justify-content-{start|center|end|between|around}
  • .align-items-{start|center|end|baseline|stretch}
  • .flex-{grow|shrink}-{0|1}
  • .order-{0-5|first|last}

Text

  • .text-{start|center|end|wrap|nowrap|break}
  • .fs-{1-6}
  • .fw-{bold|bolder|semibold|normal|light|lighter}
  • .fst-{italic|normal}
  • .text-{lowercase|uppercase|capitalize}
  • .text-decoration-{underline|line-through|none}
  • .lh-{1|sm|base|lg}

Colors

  • .text-{primary|secondary|success|danger|warning|info|light|dark|body|muted|white|black}
  • .bg-{primary|secondary|success|danger|warning|info|light|dark|white|transparent}
  • .bg-opacity-{10|25|50|75|100}
  • .bg-gradient

Sizing

  • .w-{25|50|75|100|auto}
  • .h-{25|50|75|100|auto}
  • .mw-{100}, .mh-{100}
  • .vw-{100}, .vh-{100}
  • .min-vw-{100}, .min-vh-{100}

Borders

  • .border, .border-{top|end|bottom|start|0}
  • .border-{primary|secondary|success|danger|warning|info|light|dark|white}
  • .border-{1-5}
  • .rounded, .rounded-{top|end|bottom|start|circle|pill|0-3}

Position & Shadows

  • .position-{static|relative|absolute|fixed|sticky}
  • .{top|start|bottom|end}-{0|50|100}
  • .translate-{middle|middle-x|middle-y}
  • .shadow-{none|sm|lg}
  • .z-{n1|0|1|2|3|4|5}
Remember: Add responsive prefixes to any utility class: .{sm|md|lg|xl|xxl}-{utility}

Utility-First Development Benefits

Rapid Development

Style directly in HTML without switching to CSS files.

Consistent Design

Limited design choices create visual consistency.

Reduced CSS

Smaller CSS bundle sizes and better performance.
