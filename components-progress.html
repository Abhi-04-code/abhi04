BOOTSTRAP PROGRESS BARS

▶ Progress Bars Overview

Bootstrap provides customizable progress bars to display the completion status of a task or process. They're built with HTML, CSS, and minimal JavaScript for dynamic updates.

Pro Tip: Progress bars can be styled with different colors, sizes, and can even display multiple bars stacked together.

▶ Interactive Progress Bars

JavaScript Controlled Progress

Step Progress

Step 1 of 5 
<!-- Interactive Progress Bar -->
<div class="progress mb-3">
  <div id="interactiveProgress" class="progress-bar" role="progressbar" style="width: 0%" aria-valuenow="0" aria-valuemin="0" aria-valuemax="100">0%</div>
</div>
<div class="d-flex gap-2">
  <button id="startProgress" class="btn btn-primary">Start Progress</button>
  <button id="resetProgress" class="btn btn-outline-primary">Reset</button>
</div>

<script>
document.getElementById('startProgress').addEventListener('click', function() {
  let progress = 0;
  const progressBar = document.getElementById('interactiveProgress');
  const interval = setInterval(() => {
    progress += 1;
    progressBar.style.width = progress + '%';
    progressBar.setAttribute('aria-valuenow', progress);
    progressBar.textContent = progress + '%';
    
    if (progress >= 100) {
      clearInterval(interval);
    }
  }, 50);
});

document.getElementById('resetProgress').addEventListener('click', function() {
  const progressBar = document.getElementById('interactiveProgress');
  progressBar.style.width = '0%';
  progressBar.setAttribute('aria-valuenow', 0);
  progressBar.textContent = '0%';
});
</script>

Progress Bars Best Practices

  • Provide Context: Always include labels or text to explain what the progress bar represents.
  • Use Meaningful Colors: Choose colors that match the context (green for success, red for errors, etc.).
  • Accessibility: Include ARIA attributes for screen readers and ensure sufficient color contrast.
  • Animation: Use animated stripes for processes that may take longer to complete.
