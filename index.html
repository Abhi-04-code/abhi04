BOOTSTRAP DOWNLOAD

▶ Downloading Bootstrap

There are several ways to get started with Bootstrap. You can download the compiled CSS and JS files, use a CDN, or install it via package managers like npm, yarn, or Composer.

Tip: For beginners, using the CDN method is the easiest way to get started with Bootstrap.

▶ Compiled CSS and JS

Download ready-to-use compiled code for Bootstrap v5.3.3 to easily drop into your project.

Download Options

Download Bootstrap Source Files Examples

After downloading, include the files in your project:

<!-- CSS only -->
<link href="css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript Bundle with Popper -->
<script src="js/bootstrap.bundle.min.js"></script>

▶ CDN via jsDelivr

Use the Bootstrap CDN to deliver cached version of Bootstrap's compiled CSS and JS to your project.

CSS Link

Place the following <link> in your <head> before all other stylesheets:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

JS Bundle

Place the following <script> near the end of your pages, right before the closing </body> tag:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
The JavaScript bundle above already includes Popper.js, so you don't need to include it separately.

▶ Package Managers

Install Bootstrap's source Sass and JavaScript files via npm, yarn, or Composer.

Install via npm

npm install bootstrap@5.3.3

Install via yarn

yarn add bootstrap@5.3.3

Install via Composer

composer require twbs/bootstrap:5.3.3
Note: When installing via package managers, you'll need to compile Bootstrap's Sass files yourself.

Summary: Download Methods

  • Compiled Download: Best for quick setup and prototyping.
  • CDN: Easiest for beginners and small projects.
  • Package Managers: Best for larger projects and custom builds.
