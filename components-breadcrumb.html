BOOTSTRAP BREADCRUMBS

▶ Breadcrumb Overview

Bootstrap breadcrumbs provide a navigation hierarchy for websites and applications. They indicate the current page's location within a navigational hierarchy and allow users to quickly navigate back to parent sections.

Pro Tip: Breadcrumbs are especially useful for websites with deep hierarchical structures like e-commerce sites, documentation, and file systems.

▶ Basic Breadcrumbs

Multiple Levels

▶ Breadcrumb Options

Breadcrumb Classes and Attributes

Class/Attribute Description Usage
.breadcrumb Base breadcrumb class Required for breadcrumb container
.breadcrumb-item Individual breadcrumb item Each navigation link or current page
.active Current page indicator Applied to the last breadcrumb item
aria-label="breadcrumb" Accessibility label Added to nav element for screen readers
aria-current="page" Current page indicator Added to active breadcrumb item
.rounded-pill Rounded corners Optional for custom styling

Breadcrumb Best Practices

  • Clear Hierarchy: Maintain a logical and consistent navigation structure.
  • Descriptive Labels: Use clear, concise labels for each breadcrumb level.
  • Visual Consistency: Keep breadcrumb styling consistent with your site's design system.
  • Mobile Considerations: Implement responsive behavior for smaller screens.
