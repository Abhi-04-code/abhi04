BOOTSTRAP LAYOUT COLUMNS

▶ Introduction to Columns

Columns are the fundamental building blocks of Bootstrap's grid system. They provide the structure and flexibility needed to create responsive layouts that work across all device sizes.

Bootstrap's column system is built with Flexbox, offering powerful alignment, ordering, and sizing capabilities. Understanding how to work with columns is essential for creating effective Bootstrap layouts.

Pro Tip: Bootstrap columns use a 12-column system. You can mix and match column classes to create virtually any layout, and they automatically adjust based on screen size when using responsive classes.

▶ Column Types & Classes

Basic Columns

Equal-width and specific-width columns for simple layouts.

.col .col-*
Responsive Columns

Columns that adapt to different screen sizes using breakpoints.

.col-sm-* .col-md-*
Advanced Features

Offset, ordering, alignment, and nesting capabilities.

.offset-* .order-*
Column Type Class Pattern Description Example
Equal Width .col Automatically sized equal-width columns .col
Specific Width .col-{number} Columns with specific width (1-12) .col-6
Responsive .col-{breakpoint}-{number} Width specific to breakpoint and larger .col-md-4
Auto Width .col-auto Column sized based on its content .col-auto

▶ Responsive Column System

Breakpoint-based Columns

Create responsive layouts that change at different breakpoints:

<div class="container">
  <div class="row">
    <!-- Stack on mobile, 50% on tablet, 33.3% on desktop -->
    <div class="col-12 col-sm-6 col-lg-4">
      <div class="col-item">Responsive Column</div>
    </div>
    <div class="col-12 col-sm-6 col-lg-4">
      <div class="col-item">Responsive Column</div>
    </div>
    <div class="col-12 col-sm-12 col-lg-4">
      <div class="col-item">Responsive Column</div>
    </div>
  </div>
</div>

Column Wrapping

Columns will automatically wrap to the next line when they exceed 12 units:

<div class="container">
  <div class="row">
    <div class="col-9"><div class="col-item">.col-9</div></div>
    <div class="col-4"><div class="col-item">.col-4<br>9+4=13, wraps</div></div>
    <div class="col-6"><div class="col-item">.col-6<br>4+6=10, fits</div></div>
  </div>
</div>

Column Best Practices

  • Mobile-First: Always design for mobile first, then enhance for larger screens.
  • Math Matters: Ensure columns in each row add up to 12 or fewer units.
  • Nesting: Use nested grids for complex layouts within main columns.
  • Responsive: Use responsive column classes for adaptive layouts.
