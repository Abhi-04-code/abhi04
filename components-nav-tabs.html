BOOTSTRAP NAV & TABS

▶ Navigation & Tabs Overview

Bootstrap's navigation components provide flexible and customizable navigation styles. Navigation available in Bootstrap includes tabs and pills, both with horizontal and vertical variations.

Pro Tip: Navigation components are built with flexbox and provide a strong foundation for building all types of navigation components.

▶ Data Attributes vs JavaScript

Different Ways to Activate Tabs

Data Attributes Method

This tab was activated using Bootstrap's data attributes: data-bs-toggle="tab" and data-bs-target="#tab-content".

JavaScript Method

This tab was activated using JavaScript with tab.show() method.

<!-- Data attributes method -->
<button class="nav-link active" data-bs-toggle="tab" data-bs-target="#home">
  Data Attributes
</button>

<!-- JavaScript method -->
<button class="nav-link" onclick="activateTab('profile')">
  JavaScript
</button>

<script>
function activateTab(tabId) {
  const triggerEl = document.querySelector(`#${tabId}-tab`);
  const tab = new bootstrap.Tab(triggerEl);
  tab.show();
}
</script>

Navigation & Tabs Best Practices

  • Accessibility: Use proper ARIA attributes for screen readers and keyboard navigation.
  • User Experience: Choose between tabs and pills based on your content hierarchy.
  • Visual Design: Use icons and badges to enhance navigation clarity and provide context.
  • Responsive Design: Consider vertical navigation for mobile devices or sidebar layouts.
