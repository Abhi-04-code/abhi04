BOOTSTRAP ACCORDION

▶ Accordion Overview

The Bootstrap accordion is a vertically stacked set of collapsible panels that allows users to toggle the display of content sections. It's built using the Collapse JavaScript plugin and is perfect for FAQs, content organization, and reducing visual clutter.

Pro Tip: Use the data-bs-parent attribute to create an accordion where only one panel can be open at a time.

▶ Accordion Options

Accordion Attributes and Classes

Attribute/Class Description Usage
data-bs-toggle="collapse" Enables collapse functionality Required on accordion button
data-bs-target Targets the collapsible element Points to the ID of the collapsible content
data-bs-parent Creates accordion behavior (only one open) Added to collapsible elements
aria-expanded Indicates if panel is expanded true/false on accordion button
aria-controls References the controlled element ID of the collapsible content
.accordion-flush Removes background and borders Applied to accordion container

Accordion Best Practices

  • Clear Headings: Use descriptive headings that summarize the content within each panel.
  • Logical Order: Arrange accordion items in a logical sequence that makes sense to users.
  • Visual Indicators: Use icons or color changes to indicate the active state of accordion items.
  • Performance: Avoid placing heavy content or complex components inside accordion panels.
