CUSTOMIZE BOOTSTRAP - OVERVIEW

▶ Introduction to Bootstrap Customization

Bootstrap is designed to be highly customizable. While you can use the default compiled version, the real power comes from customizing it to match your project's design system.

There are several approaches to customizing Bootstrap:

CSS Overrides

Override Bootstrap styles with your own CSS after the Bootstrap CSS file.

Easy Quick
Sass Variables

Modify Sass variables before compilation for systematic customization.

Recommended Flexible
CSS Custom Properties

Use CSS variables for runtime theming and color mode switching.

Modern Dynamic
Utility API

Generate custom utilities with Sass for project-specific needs.

Advanced Powerful

▶ Customization Workflow

Set Up Sass Environment

Install Bootstrap via npm and set up a Sass compiler in your build process.

npm install bootstrap

Create Custom Sass File

Create your main Sass file where you'll import Bootstrap and override variables.

// custom.scss
// Your variable overrides
$primary: #7C3AED;
$font-size-base: 1.1rem;

// Import Bootstrap
@import "bootstrap/scss/bootstrap";

Override Variables

Modify Sass variables to change colors, spacing, fonts, and more.

// Variable overrides
$theme-colors: (
  "primary": #7C3AED,
  "secondary": #06B6D4,
  "success": #10B981,
  "custom-color": #900
);

$border-radius: 0.5rem;
$font-family-base: 'Inter', sans-serif;

Compile Sass

Compile your Sass file to generate customized CSS.

sass custom.scss custom.css

Use Custom CSS

Link to your compiled CSS file instead of the default Bootstrap CSS.

<link href="custom.css" rel="stylesheet">

▶ Key Sass Variables to Customize

Color Variables

Variable Default Value Description
$primary #0d6efd Primary brand color
$secondary #6c757d Secondary color
$success #198754 Success state color
$danger #dc3545 Danger/error color
$warning #ffc107 Warning state color
$info #0dcaf0 Info state color

Typography & Spacing Variables

Variable Default Value Description
$font-family-base system-ui, -apple-system, ... Base font family
$font-size-base 1rem Base font size
$line-height-base 1.5 Base line height
$spacer 1rem Base spacing unit
$border-radius 0.375rem Default border radius
$border-width 1px Default border width

▶ Complete Customization Example

Custom SCSS File

// custom.scss

// 1. Include functions first
@import "bootstrap/scss/functions";

// 2. Override default variables
$primary: #7C3AED;
$secondary: #06B6D4;
$success: #10B981;
$info: #0EA5E9;
$warning: #F59E0B;
$danger: #EF4444;
$dark: #1E293B;

$font-family-base: 'Inter', system-ui, -apple-system, sans-serif;
$font-size-base: 1.1rem;
$line-height-base: 1.7;

$border-radius: 0.5rem;
$border-radius-lg: 0.75rem;
$border-radius-sm: 0.25rem;

// 3. Add custom colors to theme colors
$custom-colors: (
  "indigo": #6610f2,
  "purple": #6f42c1
);

$theme-colors: map-merge($theme-colors, $custom-colors);

// 4. Import Bootstrap
@import "bootstrap/scss/variables";
@import "bootstrap/scss/variables-dark";
@import "bootstrap/scss/maps";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/root";

// 5. Import components
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
// ... import other components as needed

// 6. Add custom utilities
$utilities: (
  "custom-shadow": (
    property: box-shadow,
    class: shadow-custom,
    values: (
      null: 0 0.5rem 1rem rgba(0, 0, 0, 0.15),
      lg: 0 1rem 3rem rgba(0, 0, 0, 0.175)
    )
  )
);

// 7. Include utility API
@import "bootstrap/scss/utilities/api";

// 8. Add custom styles
.custom-component {
  background: linear-gradient(135deg, $primary, $info);
  border-radius: $border-radius-lg;
}
This example shows a complete customization workflow with color changes, typography updates, and custom utilities.

Customization Benefits

  • Brand Consistency: Match Bootstrap components to your brand's color scheme and design language.
  • Reduced CSS: Only include the components you need, reducing file size.
  • Maintainable Code: Centralized variables make global changes easy and consistent.
  • Performance: Remove unused components and utilities to optimize performance.
