CUSTOMIZE BOOTSTRAP WITH SASS

▶ Introduction to Sass with Bootstrap

Bootstrap is built with Sass (Syntactically Awesome Style Sheets), a powerful CSS preprocessor that allows you to use variables, nested rules, mixins, functions, and more.

Using Sass with Bootstrap gives you complete control over the framework's styling and allows you to create a truly custom design system.

Pro Tip: Bootstrap 5 uses Dart Sass, which is the primary implementation of the Sass language.

▶ Setting Up Sass Environment

Install Bootstrap via npm

Start by installing Bootstrap as a dependency in your project.

npm install bootstrap

Choose a Sass Compiler

You can use any Sass compiler. Popular options include:

  • Dart Sass: The primary implementation (recommended)
  • Node Sass: Node.js binding to LibSass
  • Build Tools: Webpack, Gulp, Grunt with Sass plugins
# Install Dart Sass globally
npm install -g sass

# Or install as dev dependency
npm install --save-dev sass

Create Your Sass File Structure

Organize your Sass files for better maintainability.

your-project/
├── scss/
│   ├── _variables.scss    # Custom variables
│   ├── _custom.scss       # Additional custom styles
│   └── main.scss          # Main Sass file
├── node_modules/
│   └── bootstrap/         # Bootstrap source
└── dist/
    └── css/
        └── custom.css     # Compiled CSS

▶ Sass File Structure & Import Order

Proper Import Order

The order of imports in your main Sass file is crucial for proper compilation:

// main.scss

// 1. Include functions first (so you can manipulate colors, SVGs, calc, etc)
@import "bootstrap/scss/functions";

// 2. Include any default variable overrides here
@import "custom/variables";

// 3. Include remainder of required Bootstrap stylesheets
@import "bootstrap/scss/variables";
@import "bootstrap/scss/variables-dark";
@import "bootstrap/scss/maps";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/root";

// 4. Include any optional Bootstrap CSS as needed
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/images";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
// ... import other components as needed

// 5. Add additional custom code here
@import "custom/custom";
Important: Always import functions first, then your variable overrides, then the rest of Bootstrap. This ensures your variables take precedence.

▶ Key Sass Features in Bootstrap

Sass Variables

Bootstrap uses Sass variables for almost everything - colors, spacing, fonts, breakpoints, and more.

Color Functions

Bootstrap provides Sass functions for color manipulation:

// Lighten a color
$lighter-primary: lighten($primary, 10%);

// Darken a color  
$darker-primary: darken($primary, 10%);

// Adjust color transparency
$transparent-primary: rgba($primary, 0.8);

// Mix colors
$mixed-color: mix($primary, $secondary, 50%);
Utility Mixins

Create responsive utilities with Bootstrap's mixins:

// Create custom responsive utility
@include media-breakpoint-up(md) {
  .custom-md-padding {
    padding: 2rem;
  }
}

// Button variant mixin
.custom-button {
  @include button-variant(
    $background: $custom-color,
    $border: $custom-color,
    $color: color-contrast($custom-color)
  );
}

▶ Advanced Sass Customization

Customizing the Utility API

Bootstrap 5 introduced a Utility API that allows you to generate your own utilities:

// Add to your Sass file after Bootstrap imports

// Custom utilities
$utilities: (
  "custom-shadow": (
    property: box-shadow,
    class: shadow-custom,
    values: (
      null: 0 0.5rem 1rem rgba(0, 0, 0, 0.15),
      sm: 0 0.125rem 0.25rem rgba(0, 0, 0, 0.075),
      lg: 0 1rem 3rem rgba(0, 0, 0, 0.175)
    )
  ),
  "custom-width": (
    property: width,
    class: w-custom,
    values: (
      10: 10%,
      20: 20%,
      30: 30%,
      // Add more values as needed
    )
  )
);

// Generate the utilities
@import "bootstrap/scss/utilities/api";

Creating Custom Components

Extend Bootstrap with your own custom components using Sass:

// Custom card variant
.custom-card {
  @extend .card;
  border: 0;
  box-shadow: $box-shadow-lg;
  transition: all 0.3s ease;
  
  &:hover {
    transform: translateY(-5px);
    box-shadow: $box-shadow-xl;
  }
  
  .custom-card-header {
    @extend .card-header;
    background: linear-gradient(135deg, $primary, $info);
    color: white;
    border-bottom: 0;
  }
}

// Custom button with gradient
.btn-gradient {
  background: linear-gradient(45deg, $primary, $purple);
  border: 0;
  color: white;
  transition: all 0.3s ease;
  
  &:hover {
    transform: translateY(-2px);
    box-shadow: 0 4px 15px rgba($primary, 0.4);
    color: white;
  }
}

▶ Complete Sass Customization Example

Full Custom Sass Setup

// _variables.scss - Custom variable overrides

// Colors
$primary: #7C3AED;
$secondary: #06B6D4;
$success: #10B981;
$info: #0EA5E9;
$warning: #F59E0B;
$danger: #EF4444;
$dark: #1E293B;

// Typography
$font-family-base: 'Inter', system-ui, -apple-system, sans-serif;
$font-size-base: 1.1rem;
$headings-font-weight: 700;

// Spacing
$spacer: 1rem;
$spacers: (
  0: 0,
  1: $spacer * 0.25,
  2: $spacer * 0.5,
  3: $spacer,
  4: $spacer * 1.5,
  5: $spacer * 3,
  6: $spacer * 4.5,
  7: $spacer * 6
);

// Border radius
$border-radius: 0.5rem;
$border-radius-lg: 0.75rem;
$border-radius-sm: 0.25rem;

// Custom colors
$custom-colors: (
  "indigo": #6610f2,
  "purple": #6f42c1,
  "pink": #d63384,
  "orange": #fd7e14,
  "teal": #20c997
);

// Merge with theme colors
$theme-colors: map-merge($theme-colors, $custom-colors);

// main.scss - Main Sass file

// Functions first
@import "bootstrap/scss/functions";

// Variable overrides
@import "variables";

// Required Bootstrap imports
@import "bootstrap/scss/variables";
@import "bootstrap/scss/variables-dark";
@import "bootstrap/scss/maps";
@import "bootstrap/scss/mixins";
@import "bootstrap/scss/root";

// Optional components
@import "bootstrap/scss/utilities";
@import "bootstrap/scss/reboot";
@import "bootstrap/scss/type";
@import "bootstrap/scss/images";
@import "bootstrap/scss/containers";
@import "bootstrap/scss/grid";
@import "bootstrap/scss/buttons";
@import "bootstrap/scss/card";
// Add other components as needed

// Utility API
@import "bootstrap/scss/utilities/api";

// Custom styles
@import "custom";
This complete example shows how to set up a fully customized Bootstrap build with custom colors, typography, spacing, and components.

Sass Customization Benefits

  • Complete Design Control: Modify every aspect of Bootstrap's appearance through variables.
  • Cleaner Code: Use Sass features like variables, mixins, and functions for maintainable code.
  • Optimized Output: Include only the components you need, reducing file size.
  • Extensible: Create custom utilities and components that integrate seamlessly with Bootstrap.
