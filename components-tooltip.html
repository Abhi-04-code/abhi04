BOOTSTRAP TOOLTIP

▶ Tooltip Overview

Bootstrap Tooltips are small, contextual popups that appear when users hover over, focus on, or click an element. They provide additional information about UI elements without cluttering the interface.

Pro Tip: Tooltips must be initialized with JavaScript. For performance, initialize them selectively rather than globally if you have many tooltips on a page.

<script>
// Initialize tooltips
const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]')
const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new bootstrap.Tooltip(tooltipTriggerEl))
</script>

      data-bs-title="Manual tooltip">
  Manual control
</span>

<script>
const manualTooltip = new bootstrap.Tooltip(document.getElementById('manualTooltip'))

function showManualTooltip() {
  manualTooltip.show()
}

function hideManualTooltip() {
  manualTooltip.hide()
}

function toggleManualTooltip() {
  manualTooltip.toggle()
}
</script>

<script>
const dynamicTooltip = new bootstrap.Tooltip(document.getElementById('dynamicTooltip'))

function updateTooltipContent(newContent) {
  dynamicTooltip.setContent({ '.tooltip-inner': newContent })
}

function disableTooltip() {
  if (dynamicTooltip._isEnabled) {
    dynamicTooltip.disable()
  } else {
    dynamicTooltip.enable()
  }
}
</script>

Tooltip Best Practices

  • Accessibility: Ensure tooltips are accessible via keyboard navigation and screen readers.
  • Performance: Initialize tooltips selectively on pages with many elements to avoid performance issues.
  • Content Length: Keep tooltip content concise and avoid very long text that may not fit well.
  • Consistent Styling: Use consistent colors and styling that matches your application's design system.
  • Smart Positioning: Use auto-positioning or choose positions that don't obscure important content.
