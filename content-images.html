▶ Introduction to Bootstrap Images

Bootstrap provides several classes to style and enhance images in your web projects. These classes help with responsive behavior, thumbnails, alignment, and more.

Bootstrap's image classes work with the standard <img> element and can be combined with other utility classes for advanced styling.

Pro Tip: Always use the .img-fluid class for responsive images to ensure they scale properly on different screen sizes.

▶ Responsive Images

Fluid Images

Images in Bootstrap are made responsive with .img-fluid. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent element:

<img src="image.jpg" class="img-fluid" alt="Responsive image">
.img-fluid

This image will scale to fit its container width

Center images using margin utilities:

Responsive Alignment

Use responsive float classes to change image alignment at different breakpoints:

<!-- Float left on medium screens and up -->
<img src="..." class="float-md-start" alt="Responsive float">

<!-- Float right on large screens and up -->
<img src="..." class="float-lg-end" alt="Responsive float">

<!-- No float on small screens, left on medium and up -->
<img src="..." class="float-none float-md-start" alt="Responsive float">

▶ Picture Element & Art Direction

Responsive Images with <picture>

Use the HTML5 <picture> element for art direction or to provide alternative image formats:

<picture>
  <source srcset="..." type="image/svg+xml">
  <img src="..." class="img-fluid img-thumbnail" alt="Responsive image">
</picture>
<!-- Art direction example -->
<picture>
  <source media="(min-width: 1200px)" srcset="large-image.jpg">
  <source media="(min-width: 768px)" srcset="medium-image.jpg">
  <img src="small-image.jpg" class="img-fluid" alt="Responsive image">
</picture>

▶ Image Figures

Figure Component

Use the included .figure, .figure-img and .figure-caption classes to provide an image with an optional caption:

<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">A caption for the above image.</figcaption>
</figure>

<!-- Caption alignment -->
<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption text-end">A caption for the above image.</figcaption>
</figure>

Image Best Practices

  • Responsive: Always use .img-fluid for images to ensure they scale properly on all devices.
  • Optimization: Optimize images for web to reduce file size and improve loading times.
  • Accessibility: Always include descriptive alt text for screen readers and SEO.
  • Consistency: Use consistent image styling (borders, rounding) throughout your application.
