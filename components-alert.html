BOOTSTRAP ALERTS

▶ Alert Overview

Bootstrap alerts provide contextual feedback messages for typical user actions. They are flexible components that can contain text, links, buttons, and additional HTML elements with a variety of color options to indicate different message types.

Pro Tip: Use alerts to provide feedback that attracts the user's attention without interrupting their workflow.

▶ JavaScript Behavior

Dynamic Alert Management

<div id="alertPlaceholder"></div>
<button type="button" class="btn btn-primary" id="liveAlertBtn">Show live alert</button>

<script>
const alertPlaceholder = document.getElementById('alertPlaceholder')
const appendAlert = (message, type) => {
  const wrapper = document.createElement('div')
  wrapper.innerHTML = [
    `<div class="alert alert-${type} alert-dismissible" role="alert">`,
    `   <div>${message}</div>`,
    '   <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>',
    '</div>'
  ].join('')

  alertPlaceholder.append(wrapper)
}

const alertTrigger = document.getElementById('liveAlertBtn')
if (alertTrigger) {
  alertTrigger.addEventListener('click', () => {
    appendAlert('Nice, you triggered this alert message!', 'success')
  })
}
</script>

▶ Alert Options

Alert Classes and Attributes

Class/Attribute Description Usage
.alert Base alert class Required for all alerts
.alert-[color] Contextual color class primary, success, danger, warning, info, etc.
.alert-dismissible Enables dismiss functionality Used with close button
.fade .show Adds fade animation Used with dismissible alerts
.alert-link Styles links within alerts Applied to anchor tags
role="alert" Accessibility attribute Recommended for screen readers

Alert Best Practices

  • Clear Messaging: Use concise, actionable language that clearly communicates the purpose of the alert.
  • Appropriate Context: Match alert colors to their semantic meaning (success=green, danger=red, etc.).
  • Visual Hierarchy: Use headings and proper spacing to create clear information hierarchy within alerts.
  • Timely Display: Show alerts when relevant and provide dismiss options for non-critical messages.
