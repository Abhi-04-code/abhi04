BOOTSTRAP CONTENT TABLES

▶ Introduction to Bootstrap Tables

Bootstrap provides a clean, consistent table styling with various modifiers for different use cases. Tables are opt-in, meaning they need to be explicitly styled with Bootstrap classes.

Bootstrap's table system includes basic styling, variants, responsive behavior, and interactive features to create professional-looking data tables.

Pro Tip: Always use the .table base class to get Bootstrap's table styling. Without it, tables will use the browser's default styling.

▶ Basic Table Examples

Basic Table

Using the basic .table class adds light padding and horizontal dividers to table rows:

<table class="table">
  <thead>
    <tr>
      <th scope="col">#</th>
      <th scope="col">First</th>
      <th scope="col">Last</th>
      <th scope="col">Handle</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">1</th>
      <td>Mark</td>
      <td>Otto</td>
      <td>@mdo</td>
    </tr>
    <tr>
      <th scope="row">2</th>
      <td>Jacob</td>
      <td>Thornton</td>
      <td>@fat</td>
    </tr>
    <tr>
      <th scope="row">3</th>
      <td colspan="2">Larry the Bird</td>
      <td>@twitter</td>
    </tr>
  </tbody>
</table>

▶ Contextual Classes & Colors

Contextual Classes

Use contextual classes to color table rows or individual cells:

<!-- Colored rows -->
<tr class="table-primary">
  <th scope="row">Primary</th>
  <td>Cell</td>
  <td>Cell</td>
</tr>

<!-- Colored cells -->
<tr>
  <td class="table-success">Success</td>
  <td class="table-danger">Danger</td>
  <td class="table-warning">Warning</td>
</tr>

▶ Responsive Tables

Responsive Tables

Create responsive tables by wrapping any .table in .table-responsive to make them scroll horizontally on small devices:

<div class="table-responsive">
  <table class="table">
    ...
  </table>
</div>

▶ Table Captions & Nesting

Table Captions

A <caption> functions like a heading for a table. It helps users with screen readers to find a table and understand what it's about:

<table class="table">
  <caption>List of users</caption>
  <thead>
    ...
  </thead>
  <tbody>
    ...
  </tbody>
</table>

Nested Tables

Border styles, active styles, and table variants are not inherited by nested tables:

  <thead>
    ...
  </thead>
  <tbody>
    <tr>
      <td>
        <table class="table mb-0">
          ...
        </table>
      </td>
    </tr>
  </tbody>
</table>

Table Best Practices

  • Always use .table: Without the base table class, Bootstrap styling won't apply.
  • Responsive design: Use .table-responsive for tables that might overflow on small screens.
  • Color coding: Use contextual classes to highlight important data or status.
  • Accessibility: Use proper table structure with <thead>, <tbody>, and scope attributes.
