▶ Introduction to Bootstrap Figures

Bootstrap provides a clean, semantic way to display images with captions using the <figure> component. This is perfect for displaying images, videos, or other media with descriptive text.

The figure component includes styling for the image and caption, making it easy to create professional-looking media displays with proper semantic markup.

Pro Tip: Figures are great for blog posts, documentation, galleries, and any content where you need to pair images with descriptive captions.

▶ Figure Components & Classes

Basic Figures

Standard figure with image and caption

.figure .figure-img
Figure Alignment

Left, center, and right aligned figures

.float-start .text-center
Responsive Figures

Figures that adapt to different screen sizes

.img-fluid Grid System
Component Class Description Example
Figure Container .figure Main container for the figure component <figure class="figure">
Figure Image .figure-img Styles the image within the figure <img class="figure-img">
Figure Caption .figure-caption Styles the caption text <figcaption class="figure-caption">
Responsive Image .img-fluid Makes images scale with container .figure-img.img-fluid

▶ Basic Figure Examples

Standard Figure

Documentation and examples for displaying related images and text with the figure component in Bootstrap:

<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">A caption for the above image.</figcaption>
</figure>

Figure with Different Image Styles

You can combine figure classes with other image styling classes like rounded, thumbnail, etc.:

<!-- Rounded figure -->
<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Rounded image figure.</figcaption>
</figure>

<!-- Thumbnail figure -->
<figure class="figure">
  <img src="..." class="figure-img img-thumbnail" alt="...">
  <figcaption class="figure-caption">Thumbnail image figure.</figcaption>
</figure>

<!-- Circular figure -->
<figure class="figure">
  <img src="..." class="figure-img rounded-circle" alt="...">
  <figcaption class="figure-caption">Circular image figure.</figcaption>
</figure>

▶ Figure Alignment & Positioning

Aligning Figures

Figures can be aligned using standard text alignment classes or float utilities:

<!-- Center aligned figure -->
<figure class="figure text-center">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Center aligned figure.</figcaption>
</figure>

<!-- Right aligned figure -->
<figure class="figure text-end">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Right aligned figure.</figcaption>
</figure>

<!-- Left aligned figure -->
<figure class="figure text-start">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Left aligned figure.</figcaption>
</figure>

Floating Figures

Use float utilities to wrap text around figures:

<!-- Float left -->
<figure class="figure float-start me-3">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Floating left figure.</figcaption>
</figure>

<!-- Float right -->
<figure class="figure float-end ms-3">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Floating right figure.</figcaption>
</figure>

▶ Responsive Figures

Responsive Figure Images

Always use .img-fluid on .figure-imgs to make them responsive:

<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption">Responsive figure with fluid image.</figcaption>
</figure>
Resize your browser to see the responsive behavior

Figures in Grid System

Use Bootstrap's grid system to create responsive figure layouts:

<div class="row">
  <div class="col-sm-6 col-md-4">
    <figure class="figure">
      <img src="..." class="figure-img img-fluid rounded" alt="...">
      <figcaption class="figure-caption">Figure 1 caption.</figcaption>
    </figure>
  </div>
  <div class="col-sm-6 col-md-4">
    <figure class="figure">
      <img src="..." class="figure-img img-fluid rounded" alt="...">
      <figcaption class="figure-caption">Figure 2 caption.</figcaption>
    </figure>
  </div>
  <div class="col-sm-12 col-md-4">
    <figure class="figure">
      <img src="..." class="figure-img img-fluid rounded" alt="...">
      <figcaption class="figure-caption">Figure 3 caption.</figcaption>
    </figure>
  </div>
</div>

▶ Advanced Figure Examples

Figure with Multiple Captions

You can have multiple captions or additional content within a figure:

<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <figcaption class="figure-caption fw-bold">Main Caption</figcaption>
  <figcaption class="figure-caption text-muted">Secondary caption with additional details.</figcaption>
</figure>

Figure with Additional Content

Add buttons, badges, or other elements to your figures:

<figure class="figure">
  <img src="..." class="figure-img img-fluid rounded" alt="...">
  <div class="d-flex justify-content-between align-items-center mt-2">
    <figcaption class="figure-caption">Image with actions</figcaption>
    <div>
      <button class="btn btn-sm btn-outline-primary">Like</button>
      <button class="btn btn-sm btn-outline-secondary">Share</button>
    </div>
  </div>
</figure>

Figure Best Practices

  • Semantic HTML: Use <figure> and <figcaption> for proper semantic markup.
  • Responsive Images: Always use .img-fluid with .figure-img for responsive behavior.
  • Descriptive Captions: Use meaningful captions that add context to the image.
  • Consistent Styling: Maintain consistent figure styling throughout your application.
