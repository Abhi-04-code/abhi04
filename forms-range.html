BOOTSTRAP FORM RANGE

▶ Range Inputs

Range inputs (sliders) allow users to select a value from a continuous range. Bootstrap provides custom styling for range inputs using the .form-range class that works consistently across browsers.

Pro Tip: Use .form-range class to style your range inputs with Bootstrap's custom design. You can customize the appearance with CSS custom properties.

▶ Basic Range Input

Default Range

Current value: 50
<div class="mb-3">
  <label for="basicRange" class="form-label">Example range:</label>
  <input type="range" class="form-range" id="basicRange" min="0" max="100">
  <div class="form-text">Current value: 50</div>
</div>

▶ Range with Steps

Step-based Ranges

Current value: 50
Current value: 5.0
<!-- Step of 5 -->
<input type="range" class="form-range" id="stepRange" min="0" max="100" step="5">

<!-- Decimal steps -->
<input type="range" class="form-range" id="decimalRange" min="0" max="10" step="0.5">

▶ Range with Min/Max Values

Custom Min/Max Ranges

Current value: 0
Current value: 75%
<!-- Custom min/max -->
<input type="range" class="form-range" id="customRange" min="-50" max="50">

<!-- With default value -->
<input type="range" class="form-range" id="percentageRange" min="0" max="100" value="75">

▶ Disabled Range

Disabled Range Input

This range input is disabled and cannot be interacted with.
<div class="mb-3">
  <label for="disabledRange" class="form-label">Disabled range:</label>
  <input type="range" class="form-range" id="disabledRange" disabled>
  <div class="form-text">This range input is disabled...</div>
</div>

▶ Range with Labels

Range with Min/Max Labels

Min Max
Current volume: 50
<div class="mb-3">
  <label for="labeledRange" class="form-label">Volume control:</label>
  <div class="d-flex justify-content-between mb-1">
    <small class="form-text">Min</small>
    <small class="form-text">Max</small>
  </div>
  <input type="range" class="form-range" id="labeledRange" min="0" max="100">
  <div class="form-text text-center">Current volume: 50</div>
</div>

▶ Range with Validation

Validated Range Input

Please select a value within the range. 
<form class="was-validated">
  <div class="mb-3">
    <label for="validationRange" class="form-label">Required range (0-100):</label>
    <input type="range" class="form-range" id="validationRange" min="0" max="100" required>
    <div class="invalid-feedback">
      Please select a value within the range.
    </div>
  </div>
</form>

Range Input Best Practices

  • Clear Labels: Always provide clear labels and indicate the current value.
  • Appropriate Steps: Choose step values that make sense for your use case.
  • Value Display: Show the current value to provide feedback to users.
  • Mobile Friendly: Test range inputs on touch devices for proper usability.
