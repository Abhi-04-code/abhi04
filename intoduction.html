INTRODUCTION TO BOOTSTRAP

▶ What is Bootstrap?

Bootstrap is a free and open-source front-end framework used to create responsive and mobile-first websites quickly. It was originally developed by Twitter and is now maintained by a team of developers at GitHub.

Bootstrap includes pre-built HTML, CSS, and JavaScript components that help developers create modern, professional-looking websites without having to write everything from scratch.

Did You Know? The latest version, Bootstrap 5, removed the dependency on jQuery and introduced new utility classes and components.

▶ Why Use Bootstrap?

Mobile-First

Designed with mobile devices in mind, then scaled up for larger screens.

Fast Development

Pre-built components and utilities speed up the development process.

Customizable

Easily customize colors, spacing, and components with Sass variables.

Large Community

Huge community support with extensive documentation and resources.

Cross-Browser

Works consistently across all modern browsers and devices.

Accessibility

Built with accessibility in mind, following WCAG guidelines.

▶ Bootstrap History & Versions

Bootstrap 1 (2011)

Originally named Twitter Blueprint, created by Mark Otto and Jacob Thornton at Twitter.

Bootstrap 2 (2012)

Added 12-column responsive grid system, Glyphicons icon set, and responsive design as default.

Bootstrap 3 (2013)

Introduced mobile-first approach, flat design, and dropped IE7 support.

Bootstrap 4 (2018)

Switched from Less to Sass, added Flexbox grid, new components, and dropped IE8/IE9 support.

Bootstrap 5 (2021)

Removed jQuery dependency, added CSS custom properties, new utilities, and dropped IE10/IE11 support.

▶ Key Features of Bootstrap 5

Core Features

Feature Description
Improved Grid System Enhanced responsive grid with 6 breakpoints and improved gutter system.
Utility API Generate your own utilities with Sass for customizing spacing, colors, and more.
Custom CSS Properties CSS variables for colors, fonts, and more for easier theming.
Improved Forms Redesigned form controls with floating labels and validation styles.
No jQuery Dependency All JavaScript components rewritten in vanilla JavaScript.
New Components Added accordion, offcanvas, and placeholders components.

▶ Basic Bootstrap Template

Here's a basic HTML template to get started with Bootstrap 5:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap 5 Template</title>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <h1 class="text-center text-primary mt-5">Hello, Bootstrap!</h1>
    
    <div class="container mt-4">
        <div class="row">
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Card Title</h5>
                        <p class="card-text">This is a Bootstrap card component.</p>
                        <a href="#" class="btn btn-primary">Learn More</a>
                    </div>
                </div>
            </div>
            <div class="col-md-6">
                <div class="card">
                    <div class="card-body">
                        <h5 class="card-title">Another Card</h5>
                        <p class="card-text">Another example of Bootstrap's components.</p>
                        <a href="#" class="btn btn-outline-primary">Get Started</a>
                    </div>
                </div>
            </div>
        </div>
    </div>

    <!-- Bootstrap JS Bundle -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
This template includes the basic structure needed for a Bootstrap 5 project with a responsive grid and card components.

Summary: Why Learn Bootstrap?

  • Speed & Efficiency: Quickly build complex, consistent layouts using utility classes and pre-built components.
  • Mobile-First Responsiveness: All components and the 12-column grid are designed to work flawlessly across all screen sizes.
  • Utility-First Approach: Control almost all styling directly in your HTML with utility classes, minimizing custom CSS.
  • Industry Standard: Bootstrap is one of the most popular front-end frameworks used by developers worldwide.
