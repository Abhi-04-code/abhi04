BOOTSTRAP BUTTON GROUPS

▶ Button Group Overview

Bootstrap button groups allow you to group a series of buttons together on a single line or stack them in a vertical column. They're perfect for creating toolbars, segmented controls, and grouped navigation elements.

Pro Tip: Use button groups to create intuitive interfaces where users need to select from multiple related options, similar to radio buttons but with a more visual approach.

▶ Button Group Options

Button Group Classes and Attributes

Class/Attribute Description Usage
.btn-group Horizontal button group container Groups buttons horizontally
.btn-group-vertical Vertical button group container Stacks buttons vertically
.btn-toolbar Toolbar container for multiple groups Combines multiple button groups
.btn-group-lg / .btn-group-sm Size modifiers for button groups Controls overall group size
role="group" Accessibility role for button groups Required for screen readers
aria-label Accessibility label Describes the button group's purpose

Button Group Best Practices

  • Logical Grouping: Group related actions together to create intuitive interfaces.
  • Clear Labels: Use descriptive labels and aria attributes for accessibility.
  • Visual Consistency: Maintain consistent styling within each button group.
  • Appropriate Use: Use radio groups for single selections and checkbox groups for multiple selections.
