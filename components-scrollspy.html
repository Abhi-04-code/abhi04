BOOTSTRAP SCROLLSPY

▶ Scrollspy Overview

Bootstrap Scrollspy automatically updates navigation components based on scroll position to indicate which link is currently active in the viewport. It's perfect for long pages with multiple sections.

Pro Tip: Scrollspy requires relative positioning on the element being spied on and works with nav components or list groups.

▶ Basic Scrollspy Example

Introduction Installation Usage Customization Examples

Introduction

Scrollspy is a Bootstrap plugin that automatically updates navigation components based on scroll position. It highlights the current section in the navigation as you scroll through the page.

This is particularly useful for documentation pages, tutorials, and any content that has a clear hierarchical structure with multiple sections.

Installation

Scrollspy is included in Bootstrap's core JavaScript file, so no additional installation is required if you're using the complete Bootstrap bundle.

To use Scrollspy, you need to add specific data attributes to your HTML elements:

  • data-bs-spy="scroll" on the scrollable area
  • data-bs-target="#navbar" pointing to the navigation element
  • data-bs-offset="number" for offset adjustment

Usage

Using Scrollspy involves three main steps:

  1. Wrap your content in a scrollable container with the appropriate data attributes
  2. Create a navigation component with links that match the ID of your content sections
  3. Ensure your navigation component has the correct target ID referenced in the scrollspy

The navigation items will automatically update as you scroll through the corresponding sections.

Customization

Scrollspy can be customized in several ways:

  • Adjust the offset to account for fixed headers
  • Use different navigation components (nav, list-group, etc.)
  • Apply custom styling to active states
  • Configure scroll behavior and thresholds

You can also initialize Scrollspy via JavaScript for more control over its behavior.

Features Benefits Implementation Best Practices Troubleshooting

Features

Scrollspy offers several powerful features:

  • Automatic Activation: Navigation items are automatically activated as you scroll
  • Flexible Targeting: Works with any navigation component that contains links
  • Customizable Offset: Adjust for fixed headers and other layout elements
  • Smooth Transitions: Provides smooth scrolling between sections
  • Accessibility: Fully accessible with proper ARIA attributes

Benefits

Using Scrollspy in your projects provides several benefits:

  • Improved Navigation: Users always know where they are on the page
  • Better UX: Enhances the overall user experience for long content
  • Professional Appearance: Adds a polished, professional touch to your site
  • Easy Implementation: Simple to set up with data attributes
  • Responsive Design: Works seamlessly across all device sizes

Implementation

Implementing Scrollspy is straightforward:

  1. Include Bootstrap's JavaScript
  2. Create your navigation structure with proper IDs
  3. Add data attributes to your scrollable container
  4. Test the functionality across different screen sizes

For more complex implementations, you can use JavaScript initialization.

Best Practices

Follow these best practices for optimal Scrollspy implementation:

  • Use meaningful section IDs that match navigation labels
  • Set appropriate offset values for fixed headers
  • Test with different content lengths
  • Ensure proper contrast for active states
  • Consider mobile touch interactions

Troubleshooting

Common issues and solutions:

  • Navigation not updating: Check data-bs-target matches navigation ID
  • Incorrect active state: Verify section IDs match href values
  • Scroll position issues: Adjust data-bs-offset value
  • Performance problems: Ensure content isn't too heavy

▶ JavaScript Initialization

Programmatic Scrollspy Control

While data attributes are convenient, you can also initialize Scrollspy via JavaScript for more control:

// Initialize Scrollspy
var scrollSpy = new bootstrap.ScrollSpy(document.body, {
  target: '#navbar-scrollspy',
  offset: 100
})

// Refresh Scrollspy when content changes
scrollSpy.refresh()

// Get Scrollspy instance
var scrollSpy = bootstrap.ScrollSpy.getInstance(document.body)

// Dispose of Scrollspy instance
scrollSpy.dispose()
Configuration Options
Option Type Default Description
rootMargin string '0px 0px -25%' Margin around the root
threshold array [0.1, 0.5, 1] Visibility thresholds
smoothScroll boolean false Enable smooth scrolling

Scrollspy Best Practices

  • Proper Structure: Ensure your navigation IDs match section IDs exactly.
  • Offset Adjustment: Set appropriate offset values for fixed headers and navigation bars.
  • Mobile Considerations: Test scrollspy behavior on mobile devices with touch interactions.
  • Visual Feedback: Provide clear visual indicators for active navigation items.
  • Progressive Enhancement: Ensure content is accessible even without JavaScript.
