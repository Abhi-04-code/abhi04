BOOTSTRAP LAYOUT GUTTERS

▶ Introduction to Gutters

Gutters are the padding between your columns, used to responsively space and align content in the Bootstrap grid system. They create consistent spacing between columns while maintaining responsive behavior.

Bootstrap's gutter system is built with CSS variables, allowing for easy customization and responsive adjustments. Understanding how to work with gutters is essential for creating polished, professional layouts.

Pro Tip: Bootstrap gutters are responsive by default. They start at 1.5rem (24px) on larger screens and decrease to 1rem (16px) on smaller screens for better mobile experiences.

▶ Gutter Types & Classes

Horizontal Gutters

Spacing between columns in a row

.gx-* .g-*
Vertical Gutters

Spacing between rows in a container

.gy-* .g-*
Responsive Gutters

Gutters that adapt to different screen sizes

.gx-md-* .gy-lg-*
Gutter Type Class Pattern Description Example
Horizontal Gutters .gx-{size} Adds padding to left and right of columns .gx-3
Vertical Gutters .gy-{size} Adds padding to top and bottom of columns .gy-4
All Gutters .g-{size} Adds padding to all sides of columns .g-3
No Gutters .g-0 Removes all gutters from columns .g-0

▶ Gutter Sizes & Values

Available Gutter Sizes

Bootstrap provides a range of gutter sizes from 0 to 5, plus auto:

Size Class Value Description
0 .g-0, .gx-0, .gy-0 0rem No spacing
1 .g-1, .gx-1, .gy-1 0.25rem (4px) Extra small spacing
2 .g-2, .gx-2, .gy-2 0.5rem (8px) Small spacing
3 .g-3, .gx-3, .gy-3 1rem (16px) Medium spacing (default)
4 .g-4, .gx-4, .gy-4 1.5rem (24px) Large spacing
5 .g-5, .gx-5, .gy-5 3rem (48px) Extra large spacing

▶ Gutter Customization

Custom Gutter Sizes

You can create custom gutter sizes using CSS variables:

<style>
  .custom-gutters {
    --bs-gutter-x: 2.5rem;
    --bs-gutter-y: 1.25rem;
  }
</style>

<div class="container">
  <div class="row custom-gutters">
    <div class="col">
      <div class="gutter-highlight">Custom Gutters</div>
    </div>
    <div class="col">
      <div class="gutter-highlight">Custom Gutters</div>
    </div>
  </div>
</div>

Responsive Custom Gutters

Create responsive custom gutters using media queries:

<style>
  .responsive-custom-gutters {
    --bs-gutter-x: 1rem;
    --bs-gutter-y: 0.5rem;
  }
  
  @media (min-width: 768px) {
    .responsive-custom-gutters {
      --bs-gutter-x: 2rem;
      --bs-gutter-y: 1rem;
    }
  }
  
  @media (min-width: 1200px) {
    .responsive-custom-gutters {
      --bs-gutter-x: 3rem;
      --bs-gutter-y: 1.5rem;
    }
  }
</style>

<div class="container">
  <div class="row responsive-custom-gutters">
    <div class="col">
      <div class="gutter-highlight">Responsive Custom</div>
    </div>
    <div class="col">
      <div class="gutter-highlight">Responsive Custom</div>
    </div>
  </div>
</div>

Gutter Best Practices

  • Consistency: Use consistent gutter sizes throughout your layout for visual harmony.
  • Mobile-First: Start with smaller gutters on mobile and increase for larger screens.
  • Nesting: Use smaller gutters in nested grids to maintain hierarchy.
  • Responsive: Use responsive gutter classes for adaptive layouts across devices.
