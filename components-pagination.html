BOOTSTRAP PAGINATION

▶ Pagination Overview

Bootstrap pagination provides a set of links for navigating through multiple pages of content. It's perfect for search results, blog archives, or any content that needs to be split across multiple pages.

Pro Tip: Pagination is built with list HTML elements for screen reader accessibility and includes previous/next navigation controls.

    ▶ JavaScript Implementation

    Dynamic Pagination with JavaScript

    Current Page: 1

    <script>
// JavaScript for dynamic pagination
let currentPage = 1;
const totalPages = 10;

function generatePagination(current, total) {
    const pagination = document.getElementById('dynamicPagination');
    pagination.innerHTML = '';
    
    // Previous button
    const prevLi = document.createElement('li');
    prevLi.className = `page-item ${current === 1 ? 'disabled' : ''}`;
    prevLi.innerHTML = `<a class="page-link" href="#" onclick="changePage(${current - 1})">Previous</a>`;
    pagination.appendChild(prevLi);
    
    // Page numbers
    for (let i = 1; i <= total; i++) {
        const li = document.createElement('li');
        li.className = `page-item ${i === current ? 'active' : ''}`;
        li.innerHTML = `<a class="page-link" href="#" onclick="changePage(${i})">${i}</a>`;
        pagination.appendChild(li);
    }
    
    // Next button
    const nextLi = document.createElement('li');
    nextLi.className = `page-item ${current === total ? 'disabled' : ''}`;
    nextLi.innerHTML = `<a class="page-link" href="#" onclick="changePage(${current + 1})">Next</a>`;
    pagination.appendChild(nextLi);
}

function changePage(page) {
    if (page < 1 || page > totalPages) return;
    currentPage = page;
    document.getElementById('currentPage').textContent = page;
    document.getElementById('pageSelect').value = page;
    generatePagination(page, totalPages);
}

function goToPage(page) {
    changePage(parseInt(page));
}

// Initialize
generatePagination(currentPage, totalPages);
</script>

    Pagination Best Practices

    • Accessibility: Always include proper ARIA labels and ensure keyboard navigation works.
    • User Experience: Provide clear navigation with previous/next buttons and consider truncation for many pages.
    • Visual Design: Use active states to indicate current page and disabled states for unavailable navigation.
    • Performance: For large datasets, implement server-side pagination to limit data transfer.
