BOOTSTRAP FORM CONTROL

▶ Form Controls Overview

Form controls are the foundational elements for collecting user input in Bootstrap. The .form-control class provides consistent styling for various input types including text, email, password, textarea, and more.

Pro Tip: Use the .form-control class on textual <input> and <textarea> elements for consistent styling across all form controls.

▶ Basic Form Controls

Text Inputs

<div class="mb-3">
  <label for="exampleFormControlInput1" class="form-label">Email address</label>
  <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com">
</div>

<div class="mb-3">
  <label for="exampleFormControlInput2" class="form-label">Full Name</label>
  <input type="text" class="form-control" id="exampleFormControlInput2" placeholder="John Doe">
</div>

<div class="mb-3">
  <label for="exampleFormControlInput3" class="form-label">Password</label>
  <input type="password" class="form-control" id="exampleFormControlInput3" placeholder="Enter your password">
</div>

▶ Textarea

Multi-line Text Input

<div class="mb-3">
  <label for="exampleFormControlTextarea1" class="form-label">Example textarea</label>
  <textarea class="form-control" id="exampleFormControlTextarea1" rows="3"></textarea>
</div>

<div class="mb-3">
  <label for="exampleFormControlTextarea2" class="form-label">Large textarea</label>
  <textarea class="form-control" id="exampleFormControlTextarea2" rows="5"></textarea>
</div>

▶ Form Control Sizes

Different Input Sizes

<!-- Large -->
<input class="form-control form-control-lg" type="text" placeholder=".form-control-lg">

<!-- Default -->
<input class="form-control" type="text" placeholder="Default input">

<!-- Small -->
<input class="form-control form-control-sm" type="text" placeholder=".form-control-sm">

▶ Disabled & Readonly States

Form Control States

<!-- Disabled input -->
<input type="text" class="form-control" id="disabledInput" placeholder="Disabled input" disabled>

<!-- Readonly input -->
<input type="text" class="form-control" id="readonlyInput" value="Readonly value here..." readonly>

<!-- Disabled textarea -->
<textarea class="form-control" id="disabledTextarea" disabled>Disabled textarea content</textarea>

▶ File Input

File Upload Controls

<!-- Default file input -->
<input class="form-control" type="file" id="formFile">

<!-- Multiple files -->
<input class="form-control" type="file" id="formFileMultiple" multiple>

<!-- Disabled file input -->
<input class="form-control" type="file" id="formFileDisabled" disabled>

▶ Color Input

Color Picker

<div class="mb-3">
  <label for="exampleColorInput" class="form-label">Color picker</label>
  <input type="color" class="form-control form-control-color" id="exampleColorInput" value="#563d7c" title="Choose your color">
</div>

▶ Datalist Example

Input with Datalist

<div class="mb-3">
  <label for="exampleDataList" class="form-label">Datalist example</label>
  <input class="form-control" list="datalistOptions" id="exampleDataList" placeholder="Type to search...">
  <datalist id="datalistOptions">
    <option value="San Francisco">
    <option value="New York">
    <option value="Seattle">
    <option value="Los Angeles">
    <option value="Chicago">
  </datalist>
</div>

Form Control Best Practices

  • Consistent Styling: Always use .form-control for consistent appearance across all input types.
  • Proper Labeling: Use <label> elements with proper for attributes for accessibility.
  • Appropriate Sizing: Choose the right size (sm, lg, or default) based on your design needs.
  • Input Validation: Use appropriate input types (email, password, etc.) for built-in browser validation.
