BOOTSTRAP PLACEHOLDERS

▶ Placeholders Overview

Bootstrap placeholders can be used to enhance the user experience by displaying loading indicators for components and pages. They're built with minimal markup and styles, but they're highly customizable.

Pro Tip: Placeholders can be used to mimic the size and shape of content that is still loading, providing users with a better loading experience.

▶ Basic Placeholder

Simple Placeholder Element

<!-- Basic placeholder -->
<span class="placeholder col-6"></span>

<!-- Width utilities -->
<span class="placeholder w-75"></span>

<!-- Custom width -->
<span class="placeholder" style="width: 25%;"></span>

▶ JavaScript Implementation

Dynamic Content Loading

<script>
function loadContent() {
    const contentArea = document.getElementById('contentArea');
    const loadText = document.getElementById('loadText');
    const loadSpinner = document.getElementById('loadSpinner');
    
    // Show loading state
    loadText.textContent = 'Loading...';
    loadSpinner.classList.remove('d-none');
    
    // Simulate API call
    setTimeout(() => {
        contentArea.innerHTML = `
            <div class="alert alert-success">
                <h5>Content Loaded Successfully!</h5>
                <p>This is the actual content that has been loaded.</p>
                <p>The placeholders have been replaced with real content.</p>
            </div>
        `;
        
        loadText.textContent = 'Load Content';
        loadSpinner.classList.add('d-none');
    }, 2000);
}

function resetContent() {
    const contentArea = document.getElementById('contentArea');
    contentArea.innerHTML = `
        <div class="card-skeleton mb-3">
            <span class="placeholder col-12 mb-2"></span>
            <span class="placeholder col-10 mb-2"></span>
            <span class="placeholder col-8"></span>
        </div>
        <div class="card-skeleton">
            <span class="placeholder col-12 mb-2"></span>
            <span class="placeholder col-11 mb-2"></span>
            <span class="placeholder col-9"></span>
        </div>
    `;
}
</script>

Placeholders Best Practices

  • User Experience: Use placeholders to indicate loading states and manage user expectations.
  • Content Structure: Match placeholder dimensions to the actual content for a seamless transition.
  • Visual Design: Use animations sparingly and consider accessibility for users with motion sensitivities.
  • Performance: Implement proper loading states and error handling for dynamic content.
