BOOTSTRAP BADGES

▶ Badge Overview

Bootstrap badges are small count and labeling components that scale to match the size of the immediate parent element. They're perfect for highlighting new or unread items, displaying counts, or adding contextual labels to content.

Pro Tip: Badges automatically scale to match the size of their parent element, making them perfect for use in buttons, headings, and navigation items.

▶ Badge Options

Badge Classes and Usage

Class Description Usage
.badge Base badge class Required for all badges
.bg-[color] Background color class primary, success, danger, warning, info, etc.
.rounded-pill Makes badge pill-shaped For rounded badge style
.text-[color] Text color override Use with light backgrounds
.position-absolute For positioned badges Used with positioning utilities
.visually-hidden Accessibility for screen readers For hidden descriptive text

Badge Best Practices

  • Semantic Colors: Use color meanings appropriately (success=green, danger=red, warning=yellow).
  • Clear Labels: Keep badge text concise and meaningful for quick scanning.
  • Visual Hierarchy: Use badges to draw attention to important information without overwhelming.
  • Dynamic Content: Update badge counts in real-time for notifications and status indicators.
