BOOTSTRAP CONTENT TYPOGRAPHY

▶ Introduction to Bootstrap Typography

Bootstrap provides a clean, consistent typographic foundation with global settings for font family, sizing, spacing, and more. The typography system is built with a native font stack and uses responsive scaling.

Bootstrap's typography includes headings, body text, lists, inline text elements, and utility classes for text alignment, transformation, and styling.

Pro Tip: Bootstrap uses a native font stack that selects the best font for each OS and device, ensuring optimal readability across platforms.

▶ Text Transformation & Weight

Text Transformation

Transform text in components with text capitalization classes:

<p class="text-lowercase">Lowercased text.</p>
<p class="text-uppercase">Uppercased text.</p>
<p class="text-capitalize">CapiTaliZed text.</p>

Lowercased text - ALL LETTERS BECOME LOWERCASE.

Uppercased text - all letters become uppercase.

CapiTaliZed text - first letter of each word capitalized.

Font Weight and Italics

Quickly change the weight (boldness) of text or italicize text:

<p class="fw-bold">Bold text.</p>
<p class="fw-bolder">Bolder weight text (relative to the parent element).</p>
<p class="fw-semibold">Semibold weight text.</p>
<p class="fw-normal">Normal weight text.</p>
<p class="fw-light">Light weight text.</p>
<p class="fw-lighter">Lighter weight text (relative to the parent element).</p>
<p class="fst-italic">Italic text.</p>
<p class="fst-normal">Text with normal font style.</p>

▶ Lists & Description Lists

Unstyled List

Remove the default list-style and left margin on list items with .list-unstyled:

<ul class="list-unstyled">
  <li>This is a list.</li>
  <li>It appears completely unstyled.</li>
  <li>Structurally, it's still a list.</li>
  <li>However, this style only applies to immediate child elements.</li>
  <li>Nested lists:
    <ul>
      <li>are unaffected by this style</li>
      <li>will still show a bullet</li>
      <li>and have appropriate left margin</li>
    </ul>
  </li>
  <li>This may still come in handy in some situations.</li>
</ul>

Inline List

Remove a list's bullets and apply some light margin with .list-inline and .list-inline-item:

<ul class="list-inline">
  <li class="list-inline-item">This is a list item.</li>
  <li class="list-inline-item">And another one.</li>
  <li class="list-inline-item">But they're displayed inline.</li>
</ul>

  • And another one.
  • But they're displayed inline.

Typography Best Practices

  • Hierarchy: Use proper heading hierarchy (h1-h6) for accessibility and SEO.
  • Readability: Use appropriate line height and font sizes for optimal readability.
  • Responsive: Use responsive text alignment and sizing for different screen sizes.
  • Consistency: Maintain consistent typography throughout your application.
