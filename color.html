CUSTOMIZE BOOTSTRAP COLORS

▶ Introduction to Bootstrap Color System

Bootstrap's color system is built around a set of semantic color variables that power the entire framework. Understanding and customizing these colors is essential for creating a cohesive design system.

Bootstrap uses Sass variables and CSS custom properties (variables) to manage colors, making it easy to customize and maintain.

Pro Tip: Bootstrap 5 introduced CSS custom properties for colors, enabling runtime color changes and dark mode support.

▶ Customizing Colors with Sass

Override Theme Colors

Modify the main theme colors by overriding Sass variables before importing Bootstrap.

// Override default theme colors
$primary: #7C3AED;
$secondary: #06B6D4;
$success: #10B981;
$info: #0EA5E9;
$warning: #F59E0B;
$danger: #EF4444;
$light: #F8FAFC;
$dark: #1E293B;

// Import Bootstrap after variable overrides
@import "bootstrap/scss/bootstrap";

Add Custom Colors

Extend the theme colors map to include your custom colors.

// Define custom colors
$custom-colors: (
  "indigo": #6610f2,
  "purple": #6f42c1,
  "pink": #d63384,
  "orange": #fd7e14,
  "teal": #20c997,
  "cyan": #0dcaf0
);

// Merge with default theme colors
$theme-colors: map-merge($theme-colors, $custom-colors);

Create Color Shades

Generate lighter and darker shades of your colors using Sass functions.

// Generate color shades
$primary-100: tint-color($primary, 80%);
$primary-200: tint-color($primary, 60%);
$primary-300: tint-color($primary, 40%);
$primary-400: tint-color($primary, 20%);
$primary-600: shade-color($primary, 20%);
$primary-700: shade-color($primary, 40%);
$primary-800: shade-color($primary, 60%);
$primary-900: shade-color($primary, 80%);

// Add to color map
$theme-colors: (
  "primary-100": $primary-100,
  "primary-200": $primary-200,
  // ... add other shades
);

▶ Color Utilities & Functions

Sass Color Functions

Color Manipulation

Bootstrap provides powerful Sass functions for color manipulation:

// Lighten and darken colors
$lighter-primary: lighten($primary, 10%);
$darker-primary: darken($primary, 10%);

// Adjust saturation
$more-saturated: saturate($primary, 20%);
$less-saturated: desaturate($primary, 20%);

// Mix colors
$mixed-color: mix($primary, $secondary, 50%);

// Adjust transparency
$transparent-primary: rgba($primary, 0.5);
$with-alpha: transparentize($primary, 0.3);
Color Contrast

Automatically determine accessible text colors based on background:

// Get accessible text color for background
$text-color: color-contrast($primary);

// Usage in components
.custom-component {
  background-color: $primary;
  color: color-contrast($primary);
}

// With minimum contrast ratio
$accessible-color: color-contrast(
  $background, 
  $color-contrast-dark: #000, 
  $color-contrast-light: #fff, 
  $min-contrast-ratio: 4.5
);

▶ CSS Custom Properties (Variables)

Using CSS Color Variables

Bootstrap 5 exposes colors as CSS custom properties for runtime customization:

/* Using CSS custom properties */
.component {
  background-color: var(--bs-primary);
  color: var(--bs-white);
  border-color: var(--bs-border-color);
}

/* Overriding CSS variables */
:root {
  --bs-primary: #7C3AED;
  --bs-secondary: #06B6D4;
  --bs-success: #10B981;
}

/* For dark mode */
[data-bs-theme="dark"] {
  --bs-body-bg: #0B1220;
  --bs-body-color: #E6EEF3;
  --bs-primary: #8B5CF6;
}

Available CSS Color Variables

Variable Description Default Value
--bs-primary Primary brand color #0d6efd
--bs-secondary Secondary color #6c757d
--bs-success Success state color #198754
--bs-danger Danger/error color #dc3545
--bs-warning Warning state color #ffc107
--bs-info Info state color #0dcaf0

▶ Complete Color Customization Example

Full Color Customization with Sass

// _color-variables.scss

// 1. Custom color palette
$primary: #7C3AED;
$secondary: #06B6D4;
$success: #10B981;
$info: #0EA5E9;
$warning: #F59E0B;
$danger: #EF4444;
$light: #F8FAFC;
$dark: #1E293B;

// 2. Additional brand colors
$brand-colors: (
  "indigo": #6610f2,
  "purple": #6f42c1,
  "pink": #d63384,
  "orange": #fd7e14,
  "teal": #20c997,
  "cyan": #0dcaf0
);

// 3. Generate color shades
@function generate-shades($color) {
  $shades: ();
  $shades: map-merge($shades, ("100": tint-color($color, 80%)));
  $shades: map-merge($shades, ("200": tint-color($color, 60%)));
  $shades: map-merge($shades, ("300": tint-color($color, 40%)));
  $shades: map-merge($shades, ("400": tint-color($color, 20%)));
  $shades: map-merge($shades, ("600": shade-color($color, 20%)));
  $shades: map-merge($shades, ("700": shade-color($color, 40%)));
  $shades: map-merge($shades, ("800": shade-color($color, 60%)));
  $shades: map-merge($shades, ("900": shade-color($color, 80%)));
  @return $shades;
}

// 4. Create shade maps for each color
$primary-shades: generate-shades($primary);
$secondary-shades: generate-shades($secondary);

// 5. Merge all colors into theme colors
$custom-theme-colors: ();
$custom-theme-colors: map-merge($custom-theme-colors, $brand-colors);

// Add primary shades
@each $shade, $value in $primary-shades {
  $custom-theme-colors: map-merge($custom-theme-colors, ("primary-#{$shade}": $value));
}

// Add secondary shades  
@each $shade, $value in $secondary-shades {
  $custom-theme-colors: map-merge($custom-theme-colors, ("secondary-#{$shade}": $value));
}

// 6. Merge with default theme colors
$theme-colors: map-merge($theme-colors, $custom-theme-colors);

// main.scss
@import "bootstrap/scss/functions";
@import "color-variables";
@import "bootstrap/scss/variables";
@import "bootstrap/scss/variables-dark";
// ... rest of Bootstrap imports
This example creates a complete color system with custom colors, automatic shade generation, and integration with Bootstrap's theme system.

Color Customization Benefits

  • Brand Consistency: Match Bootstrap components exactly to your brand colors.
  • Accessibility: Ensure proper color contrast for better usability.
  • Design System: Create a cohesive color system with shades and variants.
  • Theme Support: Enable light/dark mode with CSS custom properties.
